# Tiny Epic Characters

**Tiny Epic Characters** adds one new player character model and one new monster character model configured for use with [**NightBlade MMO**](https://github.com/Fallen-Entertainment/NightBlade_MMO). The upcoming Tiny Epic Demo will use these characters, but by separating them into their own addon, you can use them in your own projects.

## Credits

- Player character model is from KayKit [Adventurers Character Pack](https://kaylousberg.itch.io/kaykit-adventurers) (CC0)
  - Model was modified to use a single skeleton so body parts can be swapped out 
  - UV texturing extended to support hair and skin colorization

- Monster character models are from KayKit [Skeletons Character Pack](https://kaylousberg.itch.io/kaykit-skeletons) (CC0)
  - Model was modified to use a single skeleton so body parts can be swapped out 

- Animations from:
  - KayKit [Character Animations](https://kaylousberg.itch.io/kaykit-character-animations) (CC0)
  - Quaternius [Universal Animation Library](https://quaternius.com/packs/universalanimationlibrary2.html) (CC0)
  - Kevin Iglesias [Sheath Script](https://assetstore.unity.com/packages/tools/animation/sheath-script-130587) (download from Unity Asset Store)

- Sounds from:
  - Duisterwho [Creature groaning](https://freesound.org/people/Duisterwho/sounds/644563/) (CC0)
